# Graphs Folder

This folder contains all graphs, charts, dashboards, metrics visualizations, and analytical outputs generated for the cybersecurity capstone project:

## Project Title

Ransomware Simulation & Recovery Strategies for Small Businesses

---

# Purpose of This Folder

The purpose of this folder is to store all graphical representations used to analyse and present project findings related to ransomware recovery performance, backup validation, and recovery efficiency.

These graphs were used throughout:

* Final project report
* PowerPoint presentation
* Jira sprint documentation
* Poster/flyer design
* Comparative analysis section

---

# Graphs Included

## Recovery Time Comparison Chart

Compares the recovery speed of Restic, Duplicati, and UrBackup after ransomware simulation.

### Metrics Included:

* Recovery duration
* Tool efficiency
* Comparative performance

---

## Recovery Success Rate Chart

Displays the percentage of successfully restored files for each backup tool.

### Metrics Included:

* Recovery success percentage
* Files restored
* Failed recovery count

## Performance Comparison Graph

# Analysis Metrics Used

The graphs in this folder were generated using the following cybersecurity recovery metrics:

| Metric                        | Purpose                          |
| ----------------------------- | -------------------------------- |
| Recovery Time Objective (RTO) | Measures recovery speed          |
| Recovery Success Rate         | Measures recovery effectiveness  |
| SHA256 Integrity Validation   | Confirms restored file integrity |
| Data Loss Percentage          | Measures recovery completeness   |
| Files Restored                | Quantifies recovery results      |
